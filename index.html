Never 13 at the table

The goal of this game is to write the different orders to avoid being 13

You can only have 12 orders at max.


To valid an order, you have to write each element down the input bar, exemple: 4 eggs

But be careful, if the total element of an order equals 13,
you must just refuse it by writing : refuse order ORDER_ID
